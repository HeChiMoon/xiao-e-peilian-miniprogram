# 小鹅智能 Agent 未来应用规划

> 版本：V1.0 | 日期：2026-05-20 | 状态：规划中

---

## 一、定位与愿景

小鹅 Agent 的长期定位：**老人膝关节健康的 AI 伴侣**——不只是回答问题，而是像一位持续陪伴的健康老师，知道老人的情况、记得聊过什么、能主动关心和提醒。

### 从"工具"到"伴侣"的三个阶段

```
Phase 1 (当前)          Phase 2 (近期)           Phase 3 (远期)
被动问答                工具调用 Agent            主动陪伴 Agent
一问一答                查数据 + 给建议            记得你、关心你、提醒你
```

---

## 二、Agent 能力演进路线

### Phase 1 — 被动问答（✅ 已完成）
- 文字 + 语音输入
- Qwen 生成回答
- TTS 语音播报
- 关键词兜底

### Phase 2 — 工具调用 Agent（🔧 当前待实现）
- 读取用户训练进度，推荐合适的动作
- 读取最近检测结果，给出个性化建议
- 查询动作标准库，给出准确步骤
- 多轮工具调用链（查进度 → 查动作 → 给方案）

### Phase 3 — 主动陪伴 Agent（📋 规划）
- 记住老人说过的情况（疼痛史、训练偏好）
- 跨会话记忆：上次聊到哪了
- 主动推送提醒（"爷爷奶奶，今天还没练哦"）
- 检测结果异常时主动发起对话
- 语音情感识别（听出老人情绪低落/焦虑）

### Phase 4 — 多模态 Agent（💡 远期）
- 老人拍照膝盖肿胀，Agent 视觉评估
- 老人录训练视频，Agent 看动作对不对
- Agent 生成教学视频片段
- 老人手指关节图说"这里疼"，Agent 理解空间位置

## 四、记忆系统设计

### 短期记忆（当前已有）
- 最近 8 轮对话历史
- 每条截断 400 字
- 会话结束即丢失

### 长期记忆（Phase 3 设计）

```
┌─────────────────────────────────────────┐
│              小鹅记忆系统                 │
├─────────────┬───────────────────────────┤
│ 用户画像    │ 年龄、性别、病史、疼痛部位    │
│ 健康档案    │ 最近检测结果、测评分数        │
│ 对话摘要    │ 上次聊了什么、关心什么        │
│ 训练日志    │ 练了哪些、频率、困难点        │
│ 疼痛日记    │ 何时疼、哪里疼、持续多久      │
│ 偏好设置    │ 喜欢文字还是语音、说话快慢     │
└─────────────┬───────────────────────────┘
              │
    voiceService 每次调用时注入到 system prompt
    （类似当前的 history，但跨会话持久化）
```

### 实现路径
1. 新建 `agent_memory` 云数据库集合
2. 每次对话结束后，用 Qwen 自动生成对话摘要
3. 下次对话开始时，注入摘要 + 用户画像到 prompt
4. Agent 开场可以主动说："爷爷奶奶，上次您提到左膝上楼梯疼，最近好点吗？"

## 五、主动推送场景

| 场景 | 触发条件 | 消息内容 |
|---|---|---|
| 训练提醒 | 今天还没完成训练 | "爷爷奶奶，今天还没练哦，花10分钟做几个直腿抬高吧。" |
| 检测跟进 | 上次检测评分偏低 | "爷爷奶奶，上次检测左膝评分偏低，这周感觉怎么样？" |
| 天气提醒 | 明天气温骤降 | "明天要降温了，膝盖可能会不舒服，记得保暖，少出门。" |
| 连续缺席 | 3 天未训练 | "爷爷奶奶，好几天没见您训练了，是膝盖不舒服还是太忙了？" |
| 疼痛恶化 | 连续 2 次记录同一部位疼痛加重 | "您最近左膝疼得比之前厉害了，小鹅建议去看看医生。" |

## 六、安全与伦理设计

### Agent 安全红线（永远不可逾越）
1. **禁止诊断**：不说"您得了XX病"
2. **禁止开药**：不推荐任何药物及剂量
3. **禁止保证**：不说"练这个一定能好"
4. **紧急情况**：立即停止建议，温柔但坚定地提醒就医
5. **身份边界**：不假装医生、家人，定位为健康科普老师

### 危险信号检测（实时触发安全回应）
- 关键词：剧痛、肿得厉害、走不了路、摔倒、发烧、关节变形、突然疼
- 连续多次描述同一疼痛且程度加重
- 老人情绪明显低落或焦虑
