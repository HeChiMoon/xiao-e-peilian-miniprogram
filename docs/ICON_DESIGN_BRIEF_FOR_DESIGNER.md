# 小鹅陪练图标设计对接说明

## 必须交付的 14 个功能图标

1. `role-elder.png`
   老人端身份选择。建议表现“老人使用小鹅陪练”或“小鹅陪老人训练”。

2. `role-caregiver.png`
   子女端身份选择。建议表现“家人看护”“手机查看长辈状态”。

3. `register-name.png`
   注册姓名输入。建议表现“填写资料”“名片”“资料卡”。

4. `voice-assistant.png`
   注册页语音助手。建议表现“小鹅 + 麦克风”，不要只画麦克风。

5. `gender-male.png`
   男性选项。建议简洁，不要过度拟人化。

6. `gender-female.png`
   女性选项。建议和男性选项成套。

7. `birth-year.png`
   出生年份选择。建议表现“日历”“年份”“时间”。

8. `profile-edit.png`
   完善资料。建议表现“资料卡 + 笔”。

9. `caregiver-bind.png`
   绑定子女。建议表现“二维码/连接/两部手机”。

10. `training-record.png`
    训练记录。建议表现“打卡记录/进度表”。

11. `health-archive.png`
    健康档案。建议表现“档案夹/健康资料”，避免医疗诊断感过强。

12. `pose-history.png`
    检测历史。建议表现“姿势骨架/检测记录/角度曲线”。

13. `settings.png`
    设置。建议温和简洁，不要做成机械齿轮感太强的科技风。

14. `scan-bind.png`
    子女端扫一扫绑定老人。建议表现“扫码 + 小鹅提示”。

## 可选补充的 4 个状态表情

1. `thinking.png`
   思考中，用于 AI 回答或加载。

2. `listening.png`
   正在听，用于语音录音。

3. `warning.png`
   温和提醒，用于动作不清楚、检测失败、安全提醒。

4. `done.png`
   完成确认，用于保存成功、绑定成功。

## 技术交付规格

源文件：

- PNG 透明背景。
- 推荐尺寸：1024 x 1024 或 1280 x 1280。
- 色彩模式：RGB。
- 文件命名使用英文小写和连字符。

小程序压缩版：

- 长边建议 220 到 260 px。
- 单张尽量控制在 120 KB 以内。
- 仍需透明背景。

如使用 Figma、PSD、AI 等源文件，也请一并提供，方便后续统一导出。
