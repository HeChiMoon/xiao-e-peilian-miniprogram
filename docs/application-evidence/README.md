本目录用于整理申报材料需要的附件截图与代码证据。最终图片和文件仍需你在申报系统中手动上传，本目录只负责归档、命名和说明。

1. AI 工具调用证明
2. 产品效果截图
3. 代码证据截图

## 截图内容要求

### 1. AI 工具调用证明

建议展示以下任意一个或多个界面：

- AI 工具对话记录
- 需求拆解对话
- 代码生成或修改对话
- 问题排查对话
- 文档整理对话

### 2. 小百科语音问答

建议展示以下任意一个或多个界面：

- 语音输入按钮
- AI 文字回答
- 小喇叭播放按钮
- 语音回答正在播放或可播放状态

### 3. 姿势检测

建议展示以下任意一个或多个界面：

- 相机检测页
- 单次检测结果页
- 检测历史页
- 检测详情页

### 4. 子女端绑定或概览

建议展示以下任意一个或多个界面：

- 扫一扫绑定页
- 绑定成功提示
- 子女端老人概览页
- 最近测评或最近检测摘要

## 代码证据建议

代码证据只截关键片段，不要截密钥、openid、个人隐私或无关调试输出。

建议文件：

- `cloudfunctions/voiceService/index.js`
- `cloudfunctions/poseService/index.js`

可截内容：

- ASR / Qwen / TTS 主流程
- TTS 缓存
- 关键点解析
- 角度计算
- 风险评分
- 检测记录保存
